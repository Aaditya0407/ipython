- 그냥 스윽 훑어보고 지나갔던 것들도 한 번 더 보게 됐고 내가 검색했던 참고 자료들을 첨부할 수도 있었고 그 첨부자료들의 양이 굉장히 방대하다. 모두 한 번에 보기는 어렵겠지만 한 번씩 훑어보면 매우 좋겠다.
